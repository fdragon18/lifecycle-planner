## 3. 計算ロジック

### 3.1 60類型番号の計算

```
G3 = MOD(シリアル日付 + 8, 60) + 1
```

- シリアル日付: Excelの日付シリアル値（例: 1988/9/23 = 32409）
- 結果: 1〜60の整数

### 3.4 派の判定

```javascript
派 = (個性番号 % 2 === 1) ? "楽観派" : "慎重派"
// 奇数(1,3,5,7,9,11) → 楽観派
// 偶数(2,4,6,8,10,12) → 慎重派
```

### 3.5 タイプの判定

個性番号からタイプコード（A★, A☆, F★, F☆, M★, M☆, Ac, Fc, Mc）を判定。

```javascript
// 推測されるマッピング（要検証）
// 個性番号 1-2: F系
// 個性番号 3-4: A系
// 個性番号 5-6: M系
// 個性番号 7-8: A系
// 個性番号 9-10: F系
// 個性番号 11-12: M系

// ★/☆の判定: 年齢または他の要素に依存
// c: 10歳未満の場合
```

### 3.6 タイプコードの生成

```
E5 = IF(年齢 < 10, "(" & タイプ & "c" & ")", "(" & タイプ & ")")
例: (A☆), (F★), (Mc)
```

## 11. 追加解析（2026-03-12）

### 11.1 *記号とサイクル文字の対応

テスト1シートで使われている記号:
- 直接使用: B, E, F, Y, Z
- *記号: *4, *5, *6, *7, *8

対応関係（数学的に検証済み - 2つの有効なマッピング）:

**マッピング1（推奨）**:
```javascript
const STAR_MAPPING = {
  '*4': 'G',
  '*5': 'X',
  '*6': 'D',
  '*7': 'A',
  '*8': 'C'
};
```

**マッピング2（代替）**:
```javascript
const STAR_MAPPING_ALT = {
  '*4': 'A',
  '*5': 'C',
  '*6': 'D',
  '*7': 'G',
  '*8': 'X'
};
```

**検証結果**: 両方のマッピングで、各SP（1-10）のサイクル順序に全10種類（A,B,C,D,E,F,G,X,Y,Z）が1回ずつ出現することを確認。

### 11.6 VBAマクロ解析結果（2026-03-12）

VBAコード（olevbaで抽出）の主要ロジック:

```vba
' 年分析 (Module1)
' 1. SPを使って行を検索
r = Sh7.Range("A32:K41").Find(What:=Sh7.Range("B3")).Row  ' B3 = SP

' 2. 起点年を使って列を検索
c = Sh7.Range("B6:K30").Find(What:=Sh7.Range("B2")).Column  ' B2 = 起点年シリアル値

' 3. r行c列から横に10個のサイクル文字を取得
Sh7.Range("B4") = Sh7.Cells(r, c)
Sh7.Range("C4") = Sh7.Cells(r, c).Next
' ... (10個のサイクル文字)
```

**ポイント**:
- ライフサイクルマスタ（年）のRow 32-41にSP(1-10)別のサイクルデータがある
- B6:K30には各年の1月1日のシリアル値（1907年〜2155年）
- SP行 × 起点年列 の交点から10個のサイクルを取得
